# UI/UX Pro Max 💎

Cette compétence déploie une intelligence de design complète pour créer des applications web et mobiles "state-of-the-art".

## 🧐 Quand utiliser cette compétence ?
Dès qu'une tâche implique de la conception visuelle, de la création de composants, du styling (Tailwind, CSS) ou de l'amélioration de l'expérience utilisateur (UX).

## 🚀 Comment l'utiliser (Workflow)

### Étape 1 : Analyse des besoins
Extraire du contexte utilisateur :
- **Type de produit** : SaaS, E-commerce, Dashboard, Landing page.
- **Style souhaité** : Minimaliste, Glassmorphism, Brutalisme, Professionnel.
- **Public cible** : Éducation, Fintech, Santé, etc.

### Étape 2 : Générer le Design System
Lancer la recherche pour obtenir des recommandations cohérentes (Couleurs, Typographie, Spacing) :

### Étape 3 : Principes de Design Premium
- **Pas d'emojis comme icônes** : Utiliser exclusivement Lucide, Heroicons ou SVG.
- **Micro-animations** : Toujours inclure `transition-all duration-300` sur les états hover.
- **Contrastes sombres/clairs** : Respecter les ratios WCAG 4.5:1.
- **Bords arrondis** : Utiliser `rounded-xl` ou `rounded-2xl` pour un look moderne.
